# ForoHub :beginner:
Este proyecto consiste en el desarrollo de una API REST para la gestión de tópicos de un foro, creada con Spring Boot. La aplicación permite realizar operaciones básicas de CRUD (crear, leer, actualizar y eliminar) sobre los tópicos.   
La API utiliza Spring Data JPA para la persistencia de datos y una base de datos MySQL para almacenar la información. Además, sigue las buenas prácticas del modelo REST para la creación de endpoints y puede ser probada utilizando herramientas como Postman.  
El objetivo del proyecto es simular el backend de un foro donde los usuarios pueden registrar y administrar temas de discusión.   

# Funcionalidades

Crear un tópico: Permite registrar un nuevo tema en el foro.

Listar tópicos: Muestra todos los tópicos registrados en el sistema.

Ver un tópico específico: Permite consultar un tópico mediante su ID.

Actualizar un tópico: Permite modificar la información de un tópico existente.

Eliminar un tópico: Permite borrar un tópico del sistema.

Estas funcionalidades están implementadas mediante una API REST desarrollada con Spring Boot, utilizando Spring Data JPA para la conexión con la base de datos MySQL.
